Як запустити проект в перший раз:

1. Підняти БД з mysqldump (знаходиться у папці backend)
2. Відкрити 2 термінали
3. Перейти у кореневу папку проекту 
4. Підтягнути та запустити backend БД за допомогою наступних команд:
   сd backend
   npm i 
   npm run start
5. Підтягнути та запустити frontend БД за допомогою наступних команд:
   cd frontend
   npm i
   npm run start
